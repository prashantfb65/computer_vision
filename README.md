# Computer vision

Filters: are essentially an image kernel, which is a small matrix applied to an entire image. Certail popular filters are well known, for example a blur filter:

[
    0.0625 0.125 0.0625
    0.125 0.25 0.125
    0.0625 0.125 0.0625
]
